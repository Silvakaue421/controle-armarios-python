# Sistema de Controle de Armários

Projeto desenvolvido em Python para gerenciamento de armários.

## Funcionalidades

- Visualizar armários disponíveis e ocupados
- Reservar armários
- Consultar informações da reserva
- Liberar armários
- Salvar dados em JSON
- Expiração automática de reservas

## Tecnologias Utilizadas

- Python
- JSON
- Datetime

## Como executar

1. Clone o repositório
2. Execute o arquivo:

```bash
python Controle_Armarios.py
